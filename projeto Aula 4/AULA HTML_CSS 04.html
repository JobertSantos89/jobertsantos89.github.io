AULA HTML/CSS 04

Nesta aula vamos aprender a utilizar o html e css de forma mais versátil para criarmos uma pagina e colocar ela no ar.
1) Primeiro passo vamos tirar um site de base para nosso projeto a ideia que vou trazer vai ser de um site para portifólio.
Este vai ser o site que vamos usar de exemplo para nosso projeto, para isso você vai precisar pegar algumas imagens no nosso google sala segue o link.
Link para imagens
Em seguida vamos criar nosso projeto no VS Code, o conselho que vou dar neste primeiro momento seria para estruturar os arquivos da seguinte forma:
Na pasta Projeto vamos ter;
uma pasta img para armazenar nossa imagens
um arquivo index.html para o codigo html
um arquivo style.css para o codigo css

No arquivo css vamos colocar 3 cores uma em cada parte eu optei por colocar verde amarelo e azul para diferenciar.
JavaScript
Copiar
main { background-color: green; .conteudo { background-color: yellow; } .imagem { background-color: blue; }
​
Agora vamos deixar nossos elemento um ao lado do outro para isso vou transformar o conteúdo de <main> em flex e centraliza-los no meio
CSS
Copiar
main { background-color: green; display: flex; justify-content: center; }
​
E também vamos aumentar a altura do nosso main para que o conteúdo fique no meio da pagina.
CSS
Copiar
main { background-color: green; display: flex; justify-content: center; height: 40vw; }
​
Feito isso vamos colocar nosso conteudo no meio de tudo para isso vamos alinhar os itens no centro
CSS
Copiar
main { background-color: green; display: flex; justify-content: center; height: 40vw; align-items: center; }
​
Estou sentindo que meu conteúdo esta muito colado na imagem vamos dar um espaçamento entre eles, para isso posso usar de duas forma:
padding: que vai aumetar o espaçamento interno entre os elemento
margin: que vai aumentar o espaçamento externo
Vamos exemplificar com os dois
Visto com os dois exemlos aplicando 10vw vamos seguir com o margin
CSS
Copiar
main>div{ margin: 10vw; }
​
Seguindo vamos deixar o nosso conteúdo mais apresentável foi criada uma classe para cada parte de elemento
HTML
Copiar
<div class="conteudo"> <!-- Conteudo --> <p class="apresentar">Olá eu sou:</p> <h1 class="nome">Otavio Augusto</h1> <p class="sobre">Programador full-stack e minhas principais habilidades são: </p> <div class="habilidades"> <img src="img/1.png" alt=""> <img src="img/2.png" alt=""> <img src="img/3.png" alt=""> <img src="img/4.png" alt=""> <img src="img/5.png" alt=""> </div> </div>
​
CSS
Copiar
.apresentar { padding-bottom: 4vh; font-size: 4vh; font-family: Verdana, Geneva, Tahoma, sans-serif; } .nome { text-align: center; padding-bottom: 4vh; font-size: 9vh; } .sobre { font-family: Verdana, Geneva, Tahoma, sans-serif; } .habilidades { display: flex; justify-content: center; height: 10vh; }
​
Feito estas alterações vamos seguir alterando o espaçamento das nossas imagens de habilidade
CSS
Copiar
.habilidades > img{ padding: 2vh; }
​
Em seguida vamos criar o efeito de quando o mouse passar por cima das imagens ele se chama hover para aplicar seguimos da seguinte forma.
CSS
Copiar
.habilidades > img:hover{ height: 60px; width: 60px; }
​

Hospedando o site

Abra sua conta no git hub e crie um novo repositorio
Suba os arquivos com o método de arrastar e soltar pasta.
Se o seu projeto está no GitHub, você pode usar o GitHub Pages para hospedar seu site gratuitamente. Aqui estão os passos básicos:
Certifique-se de que seu código está no GitHub.
No repositório do GitHub, vá para as configurações.
Role para baixo até a seção "GitHub Pages".
Escolha o branch que contém o seu código (geralmente é a branch main ou master).
O seu site estará disponível em https://seunomeusuario.github.io/nomerepositorio.
